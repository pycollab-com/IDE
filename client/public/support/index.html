Quick troubleshooting

Common fixes before you email support.

  • Refresh both clients
  • Confirm both are in the same project / session
  • Check if a browser tab was backgrounded for a long time
  • Try a different network if available
  • Reload the page and try again
  • Try a minimal test: print("hello")
  • Confirm the file has a .py extension
  • If it times out, mention runtime limits when you email support
  • Check Caps Lock is off
  • Try a private / incognito window
  • Clear site cookies for pycollab.com and retry
  • Try a different browser or disable extensions
  • Confirm the link hasn't expired — regenerate it if needed
  • Check that the room access level allows the invitee
  • Make sure the collaborator has a PyCollab account
  • Try sharing the link in a fresh message (no link-preview stripping)
